Country API Application

/addCountry

Type: POST

Create or the information of a country



/addNationality

Type: POST

Create/Update the nationality of a country and/or give a country a national dish



/getReigon

Type: GET

Get all the countries from a specific region

/getCountry

Type: GET

Get information about a single country

/getCountries

Type: GET

Gets all the countries



/getCurrencies

Type: GET

Returns all countries that use the same currency
