Countries API Documentation

/addCountry

Description:Creates a new country and adds it to the dataset.

Method Type: POST

Body Params

Country Name [required]: (String) - The name of the country

Capital [required]: (String) - The official capital of the country

Currency [required]: (String) - The official currency of the country

Returns

A JSON object that with the entered information representing a new country

Format: {'name': "ENTERED NAME", 'capital': "ENTERED CAPITAL", 'currency': "ENTERED CURRENCY"}

Examples

Request:/addCountry

Body: {"name":"Jupiter","capital":"Red Eye","currency":"Moons"}

Result: {"name":"Jupiter","capital":"Red Eye","currency":"Moons"}

Request:/addCountry

Body: {"name":"Jupiter","capital":"Red Eye","currency":""}

Result: {"message":"400 Error: Name, Capital, and Currency are all required to create country"}

/addNationality

Description:Adds/Updates unique information about a country.

Method Type: POST

Body Params

Country Name [required]: (String) - The name of the country

Nationality [required]: (String) - What residents of the country are called

National Dish [required]: (String) - The most known dish of the country

Returns

A JSON object with the entered information representing the added information

Format: {'name': "ENTERED NAME",'nationality': "ENTERED NATIONALITY", 'dish': "ENTERED DISH"}

Examples

Request:/addNationality

Context:Using /addNationality prior to adding a country

Body: {"name":"Jupiter","nationality":"Jupites","dish":"Red Eye Soup"}

Result: "404 Error: This country does not exist."

Request:/addNationality

Body: {"name":"Jupiter","nationality":"Jupites","dish":"Red Eye Soup"}

Result: {"name":"Jupiter","nationality":"Jupites","dish":"Red Eye Soup"}

Request:/addNationality

Body: {"name":"Jupiter","nationality":"Jupites","dish":""}

Result: {"message":"400 Error: Name, Nationality, and Dish are all required"}

/getRegion

Description:Returns all countries within a region of the world.

Method Type: GET & HEAD

Query Params

Continent [required]: (Dropdown Selection) - The region of the world selected to be returned

Returns

A JSON array with the names of all the countries.

Format: [{'name': 'COUNTYRY NAME'}]

Example

Request:/getRegion?region=polar

Result: [{"name":"Antarctica"}]

/getCountry

Description:Returns a single country with details about it.

Method Type: GET & HEAD

Query Params

Country Names [required]: (String) - The name of the country

Returns

A JSON array with the name, capital, natoinality, and currency of the country.

Format: [{"name":"NAME","capital":"CAPITAL","nationality":"NATIONALITY","currency":"CURRENCY"}]

Examples

Request:/getCountry?name=Italy

Result: [{"name":"Italy","capital":"Rome","nationality":"Italian","currency":"Euro"}]

Request:/getCountry?name=

Result: "400 Error: A country name was not entered"

Request:/getCountry?name=Jupiter

Result: "404 Error: This country does not exist."
